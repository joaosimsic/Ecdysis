## Crate Dependency Graph (8 crates)

```
kernel/src/main.rs (binary)
  ├─→ firehose          [Mastodon WSS + Health state machine]
  ├─→ wasm-host         [Wasmtime + WasmHost::process + gen_000 WAT]
  ├─→ cap               [Process-wide RAM ceiling]
  ├─→ arc-swap          [Hazard-pointer re-export]
  ├─→ tokio             [Async runtime]
  └─→ tracing           [Logging]
       [NOT YET IMPORTED]
       ├─→ ephemeral    [StateGraph RAM layer behind ArcSwap]
       ├─→ transpiler   [StateGraph → Rust AST via quote!]
       ├─→ evolution    [Rebirth pipeline: Synthesis→Incubation→Hot-Swap]
       ├─→ incubator    [rustc worker pool for AOT compilation]
       └─→ fsm-runtime  [Stub for M1; wasm-host handles FFI]

Relationship Key:
  • ephemeral ← transpiler (inputs state graph)
  • evolution ← transpiler, ephemeral, wasm-host, incubator (orchestrates full cycle)
  • incubator → (calls rustc on synthesized source)
  • wasm-host → (loads compiled .wasm binaries)
```

## The Missing Loop: Where M3-M8 Needs Wiring

**What should happen (per CLAUDE.md §8):**
```
FSM task loop iteration:
  1. Read byte from firehose
  2. Call WasmHost::process(byte)
     ├─ Return Terminal(NodeId) → continue
     └─ Return Unmapped(offset, byte) → Growth phase
  3. On Unmapped, allocate Box::new node in Ephemeral graph
  4. Accumulate EMA scores for edges
  5. [Periodically triggered by Harvest poll]
     └─ Call Rebirth::rebirth(graph, opts)
        ├─ Snapshot graph
        ├─ Transpile to Rust AST
        ├─ Incubate (background rustc)
        ├─ Hot-swap module
        └─ Purge institutionalized edges from RAM
```

**Current state:**
```
FSM task loop iteration:
  1. Read byte from firehose
  2. Accumulate into 64-byte buffer (NO Wasm processing)
  3. Flush buffer to Societal Bus
  4. Listen for peer frames & learn shortcuts
  [No Ephemeral, no Growth, no Harvest, no Rebirth]
```

## Summary: The Gap

**The skeleton is complete.** All individual components exist and are well-tested. But they're not wired together in `kernel/src/main.rs`.

**Missing:** `fsm_task` should:
1. Create a `WasmHost` from the live module (via evolution crate)
2. Call `host.process()` on each byte/buffer
3. On `Unmapped`, grow the Ephemeral graph
4. Periodically poll for Harvest and call `Rebirth::rebirth()`

Once that plumbing is in place, the system is theoretically alive.
